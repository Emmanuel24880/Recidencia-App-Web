# app_web_1

Descripción del Proyecto
Honorata Web, es una app que centraliza la gestión de productos, personal, pedidos, cocina y ventas diarias, optimizando la operación de negocios de alimentos y bebidas mediante una administración eficiente y organizada.

La plataforma está diseñada para facilitar el trabajo de administradores, meseros y personal de cocina mediante módulos específicos que automatizan procesos y mejoran la comunicación entre las distintas áreas del negocio.

Funcionalidades Principales

-Gestión de Productos \n
-Registro de nuevos productos.
-Edición y actualización de información.
-Activación y desactivación de productos.
-Organización por categorías.
-Control de precios, descripciones e imágenes.

Administración de Meseros

-Registro y gestión de personal.
-Asignación de accesos al sistema.
-Consulta de información de cada empleado.
-Seguimiento de actividades relacionadas con la atención al cliente.

Gestión de Órdenes

-Visualización de órdenes en tiempo real.
-Seguimiento del estado de cada pedido.
-Organización de pedidos por mesa o mesero.
-Comunicación eficiente entre meseros y cocina.

Módulo de Cocina

-Recepción automática de pedidos enviados por los meseros.
-Visualización de órdenes pendientes, en preparación y completadas.
-Actualización del estado de los alimentos durante el proceso de elaboración.
-Mejora en los tiempos de atención y entrega.

Control de Ventas

-Registro automático de ventas realizadas.
-Historial de transacciones.
-Generación de estadísticas para el análisis del negocio.

Panel Administrativo

-Monitoreo general de las operaciones.
-Acceso a información centralizada.
-Supervisión de ventas, productos y pedidos.
